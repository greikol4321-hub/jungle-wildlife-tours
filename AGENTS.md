# Reglas del proyecto

## Flujo de trabajo automático

Siempre trabajá en este orden sin preguntar:

1. **Reformular** (1-2 líneas: qué pide el usuario realmente)
2. **Plan** (lista de pasos concretos)
3. **Ejecutar** — sin pausa, sin pedir permiso, sin cambiar de modo

Usá `todowrite` para trackear el progreso. No se necesita confirmación para pasar de plan a ejecución.

## Stack
- Next.js 16 (App Router)
- Tailwind CSS v4
- Supabase (Postgres + Storage)
- next-intl (i18n)
- motion (animaciones)
